## Description

Exam project for testing competencies in Node.js and Vue.js

Usage:

Serverside:
    - Nodejs@18.14.0
    - Typesctript
    - NestJs
Clientside:
    - Nodejs@18.14.0
    - Typesctript
    - Vuejs
    - Vuex
    - Vue Router

## Running the proejct

Use docker and docker-compose

run `docker-compose up`

open url http://0.0.0.0:5000/ in browser
